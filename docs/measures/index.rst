Measures
========

Jiwar calculates various neighborhood measures for words. These measures are categorized into three main types:

.. toctree::
   :maxdepth: 2

   orthographic
   phonological
   phonographic

Overview of Measures
--------------------

Here's a quick overview of the measures calculated by Jiwar:

.. list-table::
   :header-rows: 1
   :widths: 20 30 50

   * - Measure Type
     - Measures
     - Description
   * - Orthographic
     - N, Density, OLD20, C, Neighbor Frequency
     - Based on the written form of words
   * - Phonological
     - N, Density, PLD20, C, Neighbor Frequency
     - Based on the pronunciation of words
   * - Phonographic
     - N, Density, PGLD20, C, Neighbor Frequency
     - Combines both written and spoken forms

For detailed explanations of each measure, please refer to the respective measure type pages.
